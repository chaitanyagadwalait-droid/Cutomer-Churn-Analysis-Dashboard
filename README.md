# Cutomer-Churn-Analysis-Dashboard
Customer Churn Analysis using Power BI with interactive dashboard to identify churn patterns, customer behavior, and key retention insights.

# 📊 Customer Churn Analysis Dashboard
---

#  Project Overview

Customer churn is one of the most critical challenges for subscription-based businesses. Understanding why customers leave helps companies improve retention strategies and increase revenue.

This project analyzes **customer demographics, service usage, contract types, and payment methods** to identify patterns contributing to customer churn.

Using **Power BI**, an interactive dashboard was created to help stakeholders quickly explore churn drivers and customer behavior.

---

#  Business Objective

The primary objectives of this project are:

- Identify factors influencing **customer churn**
- Analyze **customer demographics and service subscriptions**
- Evaluate churn across **contracts, internet services, and payment methods**
- Provide **data-driven insights** to improve customer retention

---

#  Dataset Information

Dataset used: **Telco Customer Churn Dataset**

Source:  
https://www.kaggle.com/datasets/blastchar/telco-customer-churn

The dataset contains **7043 customers** and multiple attributes related to customer subscription services.

### Key Columns

| Column | Description |
|------|-------------|
| CustomerID | Unique customer identifier |
| Gender | Customer gender |
| SeniorCitizen | Indicates if customer is senior |
| Partner | Whether the customer has a partner |
| Dependents | Whether the customer has dependents |
| Tenure | Number of months customer stayed |
| PhoneService | Phone subscription |
| InternetService | DSL / Fiber / No internet |
| OnlineSecurity | Security service subscription |
| DeviceProtection | Device protection plan |
| StreamingTV | Streaming TV subscription |
| StreamingMovies | Streaming movie subscription |
| Contract | Month-to-month / yearly contract |
| PaperlessBilling | Paperless billing status |
| PaymentMethod | Payment type |
| MonthlyCharges | Monthly subscription charge |
| TotalCharges | Total amount billed |
| Churn | Customer churn status |

# 📈 Key Insights

## 1️. Churn by Contract

Customers with **Month-to-Month contracts have the highest churn rate** compared to yearly contracts.

**Insight**

Encouraging customers to switch to **long-term contracts** could significantly reduce churn.

---

## 2️. Churn by Payment Method

Customers using **Electronic Check show the highest churn** compared to other payment methods.

**Insight**

Alternative payment options such as **Credit Card or Bank Transfer** may improve retention.

---

## 3️. Churn by Internet Service

Customers using **Fiber Optic Internet have the highest churn rate**.

Possible reasons:

- Higher service cost
- Customer expectations
- Network performance issues

---

## 4️. Tenure Analysis

Many customers churn within the **first few months of subscription**.

**Insight**

Improving the **early customer experience** can reduce churn.

---

## 5️. Partner & Dependents

Customers **without partners or dependents show higher churn behavior**.

**Insight**

Family-based customers may have stronger long-term service commitments.
